### 🌱 1. 설치, 환경설정 및 실행 방법
- .env 파일 생성
```
REACT_APP_CLIENT_ID=google web client id 입력
REACT_APP_REDIRECT_URI=redirect uri 입력
```
- 설치 및 실행
```
npm install
npm start
```
***
